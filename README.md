Echo Syntax:

  `echo *msg*`
 
Echov Syntax:

  `echov *varName*`
 
Clear Syntax:
  `clear *all, history, or terminal*`
 
Lock Syntax:
  `lock *history, terminal, or all*`
 
History Syntax:
  `history`
 
Var Syntax:
  `var *name* = *value*`
 
Set Syntax:
  `set *varname* = *value*`
  
Rename Syntax:
  `raname *name*`
